## Features

SortaSong offers two distinct play modes:

### üé¥ With Cards Mode (QR Scanning)

* Scan QR codes on physical cards to start a song
* Perfect for organized game sessions with prepared card decks
* Uses CameraX and ML Kit for fast, reliable QR code recognition

### üì± Without Cards Mode (Digital Deck)

* Fully digital gameplay for multiple players
* Select from available game packs
* No physical materials needed

* **Android SDK**: API Level 24 (Android 7.0) or higher

## Installation

1. Download APK from Releases and install.

## Usage

### First Launch Setup

On first launch, you'll be prompted to select a folder containing your game data.

### Data Format

The app expects a specific folder structure. The app will create folders for all games you select.
It can give you a list of songs it expects in those folders.
Songs should have the filename as Artist - Title.
The supported games and tracklists can be found in the [Available Games section](#-available-games--tracklists) below.

### Playing With Cards

1. Select "Mit Karten" from the start screen
2. Scan QR codes on your physical cards
3. The app will play the corresponding track
4. Sort tracks in your timeline by release year

### Playing Without Cards

1. Select "Ohne Karten" from the start screen
2. Add players
3. Select game packs
4. Take turns drawing and sorting cards digitally
